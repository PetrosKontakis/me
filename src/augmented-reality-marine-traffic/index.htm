Augmented Reality finder for nearby ship



This project involves the thesis that made as part of the undergraduate diploma

Thesis Title:

Design and implementation of an android mobile application detecting and exploring nearby ships built on Augmented Reality technologies.

Description:

This project is based on information extracted from Marine Traffic, regarding a ship's position and direction, worldwide. Working on that project, was given me the chance to combine two areas of my interest, augmented reality and the element of water as well. While implementing my ideas, challenges, like how a user could effectively interact with the application despite increased levels of sunshine during the day, appeared.Such requirements led me to develop an application from scratch, that includes augmented reality engine and design proposal.
